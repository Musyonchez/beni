# 1.6 Limitations of the Study

Every research project encounters constraints that may influence its scope and outcomes. The following limitations have been identified for this study, along with the measures taken to mitigate their impact:

**1. Time constraint**
The project is to be completed within a 14-week semester, which limits the depth of development and the number of features that can be fully implemented. A fully production-ready system with logistics integration, advanced analytics, and multi-language support would require significantly more time. This limitation is addressed by focusing development on a functional prototype that demonstrates the four core developer features, with additional features documented under Future Work in Chapter 6.

**2. Internet and data access among farmers**
A portion of the target farmer population in peri-urban areas of Nairobi County may have limited access to affordable mobile internet, which could affect adoption of the application. This limitation is mitigated by designing the application to be lightweight, with minimal data consumption, and by ensuring that key screens such as product listings are cached for offline viewing where possible.

**3. M-Pesa integration restrictions**
Full integration with the Safaricom Daraja API for live M-Pesa transactions requires a registered Safaricom business account, which is not accessible to individual student developers. As a result, payment integration will be demonstrated using the Daraja API sandbox environment, with the full production integration pathway documented for future deployment.

**4. Questionnaire sample size**
Access to a sufficiently large and representative sample of smallholder farmers for the study questionnaire may be constrained by geography and time. This limitation is addressed through purposive sampling targeting farmer cooperatives, Savings and Credit Cooperative Organisations (SACCOs), and local agricultural markets in the Nairobi peri-urban region, ensuring respondents are direct users of the system's intended context.
