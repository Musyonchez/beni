# FarmLink — Figures Index

All figures that appear in the report. Each figure has its own file with full
details for generating PlantUML (or other diagram tool) code.

Chapter 4 diagrams are structural (UML/DFD/ERD) — best suited for PlantUML.
Chapter 5 charts are statistical bar charts — best done in Excel, Google Sheets,
or a charting library once real questionnaire data is filled in.
