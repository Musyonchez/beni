# 2.1.4 Summary Comparison of Existing Systems

The table below presents a comparative analysis of the three reviewed systems across six key features relevant to the design and evaluation of FarmLink.

| Feature | FarmersWeb (USA) | Farmdrop (UK) | Twiga Foods (Kenya) |
|---|---|---|---|
| **Cost to farmers** | Subscription fee required | Free to list; platform takes commission | Platform sets prices; no listing fee |
| **Easy to install / access** | Web-based only; requires desktop or laptop | Web and mobile browser | Mobile app (Android); easy to install |
| **Easy to use** | Moderate; designed for business users | High; consumer-friendly interface | High; designed for low-literacy vendors |
| **Efficient database** | Yes; real-time inventory and order tracking | Yes; product catalogue and order history | Yes; large-scale supply chain data management |
| **Good security** | ✓ HTTPS, user authentication | ✓ HTTPS, secure checkout | ✓ JWT auth, encrypted transactions |
| **Vulnerability assessment** | ✓ Regular security audits reported | ✓ PCI-DSS compliant payment | ✓ Third-party security audits conducted |
| **Consumer-facing marketplace** | ✗ B2B only | ✓ Direct-to-consumer | ✗ B2B only |
| **Mobile payment integration** | ✗ Invoice-based | ✗ Card payment only | ✓ M-Pesa integration |
| **Geolocation feature** | ✗ Not available | ✗ Delivery zone-based | ✗ Not consumer-facing |
| **Seller ratings and reviews** | ✗ Not available | ✗ Not available | ✗ Not available |
| **Available in Kenya** | ✗ USA only | ✗ UK only | ✓ Kenya-based |

**Summary of findings:** None of the three reviewed systems combines all of the features required to serve the needs of Kenyan smallholder farmers and individual consumers simultaneously. FarmersWeb and Twiga Foods are B2B platforms that exclude direct consumer access, while Farmdrop — the only consumer-facing system — does not support mobile payments, geolocation, or seller ratings, and is no longer operational. FarmLink is positioned to address these collective gaps by providing a consumer-facing, mobile-first eMarketplace with M-Pesa integration, geolocation-based farmer discovery, real-time order tracking, and a seller rating system — all tailored to the Kenyan context.
