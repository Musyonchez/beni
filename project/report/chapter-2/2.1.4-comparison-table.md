# 2.1.4 Summary Comparison of Existing Systems

The table below presents a comparative analysis of the three reviewed systems across key features relevant to the design and evaluation of FarmLink.

| Feature | FarmDrop (USA) | FarmGate Africa (Nigeria) | Mkulima Online (Kenya) |
|---|---|---|---|
| **Cost to farmers** | 5–15% platform fee; farmers keep 85–95% | Commission on closed deals | Free or low-cost basic listing |
| **Easy to install / access** | Web-based; community hub required | Web-based; B2B registration | Web-based; WhatsApp communication |
| **Easy to use** | Moderate; hub manager coordinates | Moderate; designed for cooperatives | High; familiar WhatsApp-based flow |
| **Efficient database** | Yes; inventory and order tracking | Yes; commodity listing management | Moderate; listing-based, no order DB |
| **Good security** | ✓ HTTPS, user authentication | ✓ HTTPS, user authentication | ✓ HTTPS; off-platform payments |
| **Vulnerability assessment** | ✓ Regular platform updates (2025) | Partial; startup-stage security | Partial; no payment data handled |
| **Consumer-facing marketplace** | ✓ Direct-to-consumer | ✗ B2B only | ✓ C2C and B2C |
| **Mobile payment integration** | ✗ No integrated payment | ✗ Off-platform payment | ✗ Off-platform (WhatsApp/cash) |
| **Geolocation feature** | ✗ Hub-location based only | ✗ Not available | ✗ Not available |
| **Seller ratings and reviews** | ✗ Not available | ✗ Not available | Partial; verified seller badge only |
| **Real-time order tracking** | ✗ Not available | ✗ Not available | ✗ Not available |
| **Available in Kenya** | ✗ USA only | ✗ Nigeria only | ✓ Kenya-based |
| **Operational as of 2026** | ✓ Active | ✓ Active | ✓ Active, daily listings |

**Summary of findings:** The three reviewed systems collectively demonstrate that digital agricultural marketplaces are viable and actively used across different market contexts. However, a clear and consistent gap emerges across all three platforms: none of them combines integrated mobile payment processing, geolocation-based farmer discovery, real-time order tracking, and a seller rating system within a single consumer-facing mobile application. FarmDrop excels at fair farmer pricing but lacks mobile infrastructure. FarmGate Africa addresses commodity aggregation but excludes individual consumers. Mkulima Online is the closest contextual match to FarmLink's target market, yet relies on off-platform payments and WhatsApp negotiation rather than a structured, trustworthy transaction environment. FarmLink is designed to close these gaps by delivering all four missing features within a single mobile-first application built specifically for the Kenyan agricultural context.
