## Relationships Summary

| Relationship | Multiplicity | Description |
|---|---|---|
| Farmer → Product | 1 to many | A farmer creates many product listings |
| Buyer → Order | 1 to many | A buyer places many orders |
| Farmer → Order | 1 to many | A farmer receives many orders |
| Order → Review | 1 to 1 | Each completed order has at most one review |
| User → Notification | 1 to many | A user receives many notifications |
