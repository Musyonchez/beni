# 4.0 Introduction

This chapter presents the systems analysis and design of FarmLink — an online agricultural produce eMarketplace for Kenyan smallholder farmers. It begins by identifying the functional and non-functional requirements of the system, followed by a description of the system's key stakeholders. The chapter then presents a comprehensive set of system models including a systems architecture diagram, use case diagram, flowchart, class diagram, data flow diagrams at three levels, an entity relationship diagram, and sequence diagrams for both the administrator and user flows. Together, these models provide a complete blueprint of the FarmLink system that guided the development of the prototype documented in Chapter Seven.
