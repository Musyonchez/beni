# 4.5 Resource Requirements

## 4.5.1 Hardware Requirements

**Development Hardware** (Already Owned - $0 additional cost)
- **Primary Development Machine**: Laptop with minimum specifications:
  - Processor: Intel Core i5 (8th gen+) or equivalent AMD Ryzen
  - RAM: 8GB minimum (16GB preferred for emulator performance)
  - Storage: 256GB SSD minimum (Flutter + Android Studio ~20GB, project files ~5GB)
  - Operating System: Windows 10/11, macOS 10.14+, or Linux (Ubuntu 18+)
  
**Testing Devices** (Already Owned - $0 additional cost)
- **Primary Test Device**: Android smartphone (personal device)
  - Android 8.0 (Oreo) or higher
  - Minimum 2GB RAM for smooth app performance
  - Active M-Pesa account for payment testing
- **Optional Secondary Device**: Borrowed/lab iOS device for cross-platform verification

**Infrastructure** (Cloud-based - included in free tiers)
- Firebase hosting (no physical server required)
- Cloud database (Firebase Firestore - managed service)

**Total Hardware Cost**: $0 (utilizing existing personal equipment)

## 4.5.3 Human Resources

**Solo Developer** (Student - No Additional Cost)
- **Role**: Full-stack mobile developer, designer, tester, project manager
- **Time Commitment**: 15-20 hours/week × 10 weeks = 150-200 total hours
- **Skills Required**:
  - Flutter/Dart development (to be learned during project)
  - Firebase backend integration
  - M-Pesa API integration
  - UI/UX design basics
  - Project management and documentation

**Skill Development Plan**:
- **Week 1**: Flutter fundamentals (online courses, documentation)
- **Week 2-3**: Firebase integration tutorials
- **Week 4+**: Learn-by-doing during implementation

**External Support** (No Cost)
- **Project Supervisor**: Academic guidance and milestone reviews
- **Peer Reviewers**: Classmates for code review and feedback
- **User Testers**: Student volunteers (15+ participants for testing phase)

**No hiring or outsourcing planned** - entire project completed solo within academic context.
