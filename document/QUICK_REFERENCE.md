2. **Save versions** - Use dates in filenames
3. **Cite as you write** - Don't leave it for later
4. **Use templates** - Follow the READMEs strictly
5. **Get feedback early** - Show drafts to supervisor

### Development
1. **Document as you build** - Take screenshots immediately
2. **Save code snippets** - Copy important code sections
3. **Track challenges** - Note problems when they occur
4. **Version control** - Commit regularly with good messages
5. **Test continuously** - Don't wait until the end

### Time Management
1. **Set deadlines** - Personal deadlines before official ones

3. **Prioritize** - Core chapters before polish
