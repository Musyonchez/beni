# Front Matter

This folder contains all the preliminary sections that appear before the main body of the report.

## Files to Create

### 1. cover-page.md
- Student name and ID
- School name
- Course code
- Project title
- **Target Word Count**: ~30 words

### 2. declaration.md
- Statement of originality
- Student signature section
- Supervisor signature section
- **Target Word Count**: ~180 words

### 3. acknowledgement.md
- Gratitude to God
- Thanks to supervisor
- Thanks to faculty and staff
- Thanks to classmates and family
- **Target Word Count**: ~150 words

### 4. abstract.md
- Project overview (2-3 sentences)
- Problem addressed (2-3 sentences)
- Solution approach and technologies (2-3 sentences)
- Methodology (2-3 sentences)
- Expected outcomes (2-3 sentences)
- **Target Word Count**: ~200 words

### 5. table-of-contents.md
- Auto-generated from chapter structure
- Include list of figures
- Include list of tables
- **Target Word Count**: ~800 words (mostly headings)

## Notes
- Use formal, academic tone
- Third-person perspective
- Maintain professional formatting
- All content should be original to your project
