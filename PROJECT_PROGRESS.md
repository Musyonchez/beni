### System Features
Your Cafeteria Ordering & Loyalty Reward system will include:

#### Customer Features
- Browse cafeteria menu with real-time availability
- Add items to shopping cart
- Place orders in advance
- Multiple payment options (digital wallets, cards)
- Track order status with notifications
- Earn loyalty points for purchases
- Redeem points for rewards/discounts
- View order history

#### Cafeteria Staff Features
- Receive and manage orders
- Update menu items and availability
- Process orders and mark as ready
- View analytics and reports

- Manage users and permissions
- Configure loyalty program rules
- View business analytics
- Manage menu and pricing

- **Payments**: M-Pesa API / Stripe / PayPal
